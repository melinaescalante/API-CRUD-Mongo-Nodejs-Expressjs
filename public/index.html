API de Zapatillas

¿Que brindamos?

Bienvenido a nuestra API REST, un recurso completo para acceder a información detallada sobre zapatillas y sus respectivas marcas. Esta API ha sido desarrollada para facilitar la consulta de datos estructurados sobre distintos modelos de zapatillas, incluyendo características clave como nombre, marca, y descripción específica de cada producto. Ideal para desarrolladores, comerciantes y entusiastas, nuestra API te permite integrar esta información en tus aplicaciones, sitios web o análisis de mercado. Con endpoints sencillos y eficientes, podrás realizar búsquedas rápidas y obtener respuestas claras que se adapten a las necesidades de tu proyecto.

EndPoints

Users

  • GET http://127.0.0.1:3000/api/users
  • GET by id http://127.0.0.1:3000/api/users/:id
  • GET by name http://127.0.0.1:3000/api/users/name/:name
  • POST http://127.0.0.1:3000/api/users
  • POST login http://127.0.0.1:3000/api/users/login
  • PUT http://127.0.0.1:3000/api/users/:id
  • DELETE http://127.0.0.1:3000/api/users/:id

Zapatillas

  • GET http://127.0.0.1:3000/api/sneakers
  • GET by id http://127.0.0.1:3000/api/sneakers/:id
  • GET by model http://127.0.0.1:3000/api/sneakers/model/:name
  • GET by brand http://127.0.0.1:3000/api/sneakers/brand/:brand
  • POST http://127.0.0.1:3000/api/sneakers
  • PUT http://127.0.0.1:3000/api/sneakers/:id
  • DELETE http://127.0.0.1:3000/api/sneakers/:id

Marcas

  • PUT http://127.0.0.1:3000/api/brands/:id
